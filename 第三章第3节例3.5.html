三重积分的换元法

直角坐标  ·  柱面坐标  ·  球面坐标   |   一题多解，融会贯通

例 3.5

计算三重积分

$$\iiint_{\Omega} z\,\mathrm{d}V$$

其中 $\Omega$ 是由锥面 $z = \sqrt{x^2+y^2}$ 与球面 $x^2+y^2+z^2 = 2z$ 所围成的区域.

1
直角坐标法

解题思路：将三重积分化为三次积分进行计算. 先确定积分区域 $\Omega$ 在 $xOy$ 平面上的投影.

1
分析积分区域：
锥面 $z = \sqrt{x^2+y^2}$.
球面 $x^2+y^2+z^2 = 2z$ 可化为 $x^2+y^2+(z-1)^2 = 1$，即球心在 $(0,0,1)$、半径为 $1$ 的球面.
2
求两曲面的交线：
联立 $z = \sqrt{x^2+y^2}$ 和 $x^2+y^2+z^2 = 2z$，代入得 $2z^2 = 2z$，解得 $z = 1$（$z = 0$ 不在锥面上方舍去），于是 $x^2+y^2 = 1$.
交线为圆：$z = 1,\ x^2+y^2 = 1$.
3
确定投影区域 $D_{xy}$：
将锥面 $z = \sqrt{x^2+y^2}$ 代入球面方程，消去 $z$，得 $x^2+y^2 = 1$.
因此 $D_{xy}: x^2+y^2 \leqslant 1$.
4
确定 $z$ 的积分限：
对于 $D_{xy}$ 中任意一点 $(x,y)$，$z$ 从下曲面（锥面 $z = \sqrt{x^2+y^2}$）到上曲面（球面 $z = 1+\sqrt{1-x^2-y^2}$）.
即 $\sqrt{x^2+y^2} \leqslant z \leqslant 1+\sqrt{1-x^2-y^2}$.

建立累次积分

区域 $D_{xy}$ 是半径为 $1$ 的圆盘，使用极坐标变换：

$$I = \iint_{D_{xy}}\mathrm{d}x\,\mathrm{d}y\int_{\sqrt{x^2+y^2}}^{1+\sqrt{1-x^2-y^2}} z\,\mathrm{d}z = \int_0^{2\pi}\mathrm{d}\theta\int_0^1 r\,\mathrm{d}r\int_r^{\sqrt{2-r^2}} z\,\mathrm{d}z.$$
注意：球面方程 $x^2+y^2+(z-1)^2=1$ 化为 $z = 1+\sqrt{1-r^2}$（上半球面）

计算积分

(1)
先对 $z$ 积分： $$\int_r^{\sqrt{2-r^2}} z\,\mathrm{d}z = \left[\frac{z^2}{2}\right]_r^{\sqrt{2-r^2}} = \frac{2-r^2}{2} - \frac{r^2}{2} = 1-r^2.$$
(2)
再对 $r$ 积分： $$\int_0^1 r(1-r^2)\,\mathrm{d}r = \int_0^1 (r-r^3)\,\mathrm{d}r = \left[\frac{r^2}{2}-\frac{r^4}{4}\right]_0^1 = \frac{1}{2}-\frac{1}{4} = \frac{1}{4}.$$
(3)
最后对 $\theta$ 积分： $$\int_0^{2\pi}\frac{1}{4}\,\mathrm{d}\theta = \frac{1}{4}\cdot 2\pi = \frac{\pi}{2}.$$
$$\iiint_{\Omega} z\,\mathrm{d}V = \boxed{\dfrac{\pi}{2}}.$$

直角坐标：积分区域 $\Omega$ 的示意图

2
柱面坐标法

核心思想：利用柱面坐标 $x = r\cos\theta,\ y = r\sin\theta,\ z = z$ 简化计算. 此区域在 $xOy$ 平面的投影是圆盘 $r \leqslant 1$，非常适合柱面坐标.

1
柱面坐标变换：
$$x = r\cos\theta,\quad y = r\sin\theta,\quad z = z.$$
体积微元 $\mathrm{d}V = r\,\mathrm{d}r\,\mathrm{d}\theta\,\mathrm{d}z$，被积函数 $z = z$.
2
确定坐标范围：
$\theta$：由区域关于 $z$ 轴对称，$\theta: 0 \to 2\pi$.
$r$：投影区域是单位圆，$r: 0 \to 1$.
$z$：从锥面 $z = r$ 到球面 $z = \sqrt{2-r^2}$（上半球面），$r \to \sqrt{2-r^2}$.

建立累次积分

$$I = \iiint_{\Omega} z\,\mathrm{d}V = \int_0^{2\pi}\mathrm{d}\theta\int_0^1 r\,\mathrm{d}r\int_r^{\sqrt{2-r^2}} z\,\mathrm{d}z.$$
注意：球面方程 $x^2+y^2+z^2 = 2z$ 在柱面坐标下为 $r^2+z^2 = 2z$，即 $z = \sqrt{2-r^2}$

计算积分（利用积分可分离）

(1)
先对 $z$ 积分： $$\int_r^{\sqrt{2-r^2}} z\,\mathrm{d}z = \left[\frac{z^2}{2}\right]_r^{\sqrt{2-r^2}} = \frac{2-r^2}{2} - \frac{r^2}{2} = 1-r^2.$$
(2)
再对 $r$ 和 $\theta$ 积分（注意可分离）： $$\int_0^{2\pi}\mathrm{d}\theta\int_0^1 r(1-r^2)\,\mathrm{d}r = \left(\int_0^{2\pi}\mathrm{d}\theta\right)\left(\int_0^1 (r-r^3)\,\mathrm{d}r\right) = 2\pi\cdot\frac{1}{4} = \frac{\pi}{2}.$$
$$\iiint_{\Omega} z\,\mathrm{d}V = \boxed{\dfrac{\pi}{2}}.$$
关键技巧：柱面坐标下，积分限的确定非常简洁. 投影区域 $r \leqslant 1$ 直接给出 $r: 0 \to 1$，无需像直角坐标那样计算复杂的投影边界.

柱面坐标：积分区域 $\Omega$ 及坐标面示意

3
球面坐标法

核心思想：球面坐标 $\rho$ 表示到原点的距离，$\varphi$ 表示与 $z$ 轴的夹角，$\theta$ 表示方位角. 体积微元 $\mathrm{d}V = \rho^2\sin\varphi\,\mathrm{d}\rho\,\mathrm{d}\varphi\,\mathrm{d}\theta$.

1
球面坐标变换：
$$x = \rho\sin\varphi\cos\theta,\quad y = \rho\sin\varphi\sin\theta,\quad z = \rho\cos\varphi.$$
体积微元 $\mathrm{d}V = \rho^2\sin\varphi\,\mathrm{d}\rho\,\mathrm{d}\varphi\,\mathrm{d}\theta$，被积函数 $z = \rho\cos\varphi$.
2
将边界曲面化为球面坐标形式：
锥面 $z = \sqrt{x^2+y^2}$：$\rho\cos\varphi = \sqrt{\rho^2\sin^2\varphi}$，即 $\tan\varphi = 1$，故 $\varphi = \dfrac{\pi}{4}$.
球面 $x^2+y^2+z^2 = 2z$：$\rho^2 = 2\rho\cos\varphi$，即 $\rho = 2\cos\varphi$（$\rho > 0$）.
3
确定坐标范围：
$\theta$：由区域关于 $z$ 轴对称，$\theta: 0 \to 2\pi$.
$\varphi$：锥面方程 $z = \sqrt{x^2+y^2}$ 对应 $\varphi = \dfrac{\pi}{4}$（与 $z$ 轴夹角）。
    由于 $z = \rho\cos\varphi$ 且 $r = \rho\sin\varphi$，锥面 $z = r$ 等价于 $\cos\varphi = \sin\varphi$，即 $\varphi = \dfrac{\pi}{4}$.
    在锥面上方（$z \geqslant r$，即 $\cos\varphi \geqslant \sin\varphi$）对应 $\varphi \in \left[0, \dfrac{\pi}{4}\right]$.
$\rho$：从原点（$\rho = 0$）到球面（$\rho = 2\cos\varphi$），故 $\rho: 0 \to 2\cos\varphi$.

建立累次积分

$$I = \iiint_{\Omega} z\,\mathrm{d}V = \int_0^{2\pi}\mathrm{d}\theta\int_0^{\pi/4}\mathrm{d}\varphi\int_0^{2\cos\varphi} \rho\cos\varphi\cdot\rho^2\sin\varphi\,\mathrm{d}\rho.$$
被积函数 $z = \rho\cos\varphi$，体积微元 $\mathrm{d}V = \rho^2\sin\varphi\,\mathrm{d}\rho\,\mathrm{d}\varphi\,\mathrm{d}\theta$

计算积分

(1)
对 $\rho$ 积分： $$\int_0^{2\cos\varphi} \rho^3\cos\varphi\sin\varphi\,\mathrm{d}\rho = \cos\varphi\sin\varphi\cdot\left[\frac{\rho^4}{4}\right]_0^{2\cos\varphi} = \cos\varphi\sin\varphi\cdot\frac{(2\cos\varphi)^4}{4} = 4\cos^5\varphi\sin\varphi.$$
(2)
对 $\varphi$ 积分（令 $u = \cos\varphi$，$\mathrm{d}u = -\sin\varphi\,\mathrm{d}\varphi$）：
当 $\varphi = 0$ 时 $u = 1$；当 $\varphi = \dfrac{\pi}{4}$ 时 $u = \dfrac{\sqrt{2}}{2}$. $$\int_0^{\pi/4} 4\cos^5\varphi\sin\varphi\,\mathrm{d}\varphi = 4\left[-\frac{\cos^6\varphi}{6}\right]_0^{\pi/4} = \frac{4}{6}\left(1 - \left(\frac{\sqrt{2}}{2}\right)^6\right).$$ 由于 $\left(\dfrac{\sqrt{2}}{2}\right)^6 = \dfrac{1}{8}$， $$\frac{4}{6}\left(1 - \frac{1}{8}\right) = \frac{4}{6}\cdot\frac{7}{8} = \frac{7}{12}.$$
(3)
对 $\theta$ 积分： $$\int_0^{2\pi}\frac{7}{12}\,\mathrm{d}\theta = \frac{7}{12}\cdot 2\pi = \frac{7\pi}{6}.$$
结果异常！球面坐标法得 $\dfrac{7\pi}{6} \approx 3.665$，而柱面坐标法得 $\dfrac{\pi}{2} \approx 1.571$，两者不一致！
这说明球面坐标法的计算过程存在错误，需仔细检查...

深入分析：为何球面坐标给出了错误结果？

检查关键点：$z$ 轴（$x = y = 0$）上的点 $(0,0,z)$ 是否在区域 $\Omega$ 内？

锥面约束 $z = \sqrt{x^2+y^2}$ 在 $z$ 轴上

当 $x = y = 0$ 时，锥面方程 $z = \sqrt{0} = 0$，即 只有原点 $(0,0,0)$ 满足锥面方程.

对于 $z$ 轴上 $z > 0$ 的点，锥面要求 $z \leqslant 0$（因为 $z = r \geqslant 0$ 而 $r = 0$），因此 $z > 0$ 的 $z$ 轴点均不在锥面以内.

但在柱面坐标中，$r \in [0,1]$ 描述的是 $xy$ 平面上的投影区域，并非 $z$ 轴上的点！

球面坐标积分的 $\rho$ 范围分析

球面方程 $\rho = 2\cos\varphi$ 的来源：$\rho^2 = 2\rho\cos\varphi$，化简得 $\rho = 2\cos\varphi$（要求 $\rho > 0$，即 $\cos\varphi > 0$，$\varphi \in (-\pi/2, \pi/2)$）。

在 $\varphi \in \left[0, \dfrac{\pi}{4}\right]$ 范围内，$\rho = 2\cos\varphi \in \left[\sqrt{2}, 2\right]$.

这意味着在 $\varphi = 0$（$z$ 轴）方向上，$\rho$ 从 $0$ 到 $2$，对应的 $z = \rho\cos\varphi$ 从 $0$ 到 $2$，覆盖了整段 $z$ 轴 $[0,2]$.

但根据锥面方程，$z$ 轴上只有原点才满足锥面！因此 球面坐标积分多包含了 $z$ 轴上除原点外的所有点，这导致了错误的结果.

根本原因：本例题中，锥面 $z = \sqrt{x^2+y^2}$ 的顶点（原点）不在 $z$ 轴的积分路径上（除 $\rho = 0$ 外），导致球面坐标积分的 $\rho$ 上限范围包含了本不属于区域 $\Omega$ 的点.

更精确地说，球面坐标 $\rho = 2\cos\varphi$ 描述的是以原点为起点的射线与球面的交点。但在 $\varphi \in \left[0, \dfrac{\pi}{4}\right]$ 的范围内，当 $\varphi = 0$ 时，$\rho$ 从 $0$ 到 $2$ 沿 $z$ 轴方向，但锥面在 $z$ 轴上只允许 $z = 0$ 的点，导致这段区域（$0 < z \leqslant 2$）不应被包含在积分中.

正确做法：柱面坐标是本题最优解

由于柱面坐标的 $r$ 坐标独立于 $z$，能够精确描述投影区域 $r \leqslant 1$，不受锥面顶点的影响，因此是本题的最佳选择.

正确结果：

$$\iiint_{\Omega} z\,\mathrm{d}V = \boxed{\dfrac{\pi}{2}}.$$
柱面坐标验证一致，球面坐标因锥顶位于原点导致积分范围错误

为什么柱面坐标最简便？

本例题中柱面坐标的独特优势
  • 投影区域简洁：积分区域 $\Omega$ 在 $xOy$ 平面的投影是单位圆 $r \leqslant 1$，这正是柱面坐标的"自然"投影.
  • 边界方程简单：锥面 $z = \sqrt{x^2+y^2}$ 化为 $z = r$，球面 $x^2+y^2+z^2 = 2z$ 化为 $z = \sqrt{2-r^2}$，都是 $r$ 的一元函数.
  • 被积函数自然：$z = z$ 在柱面坐标下形式不变，无需复杂变换.
  • Jacobi 行列式简单：$|J| = r$，形式简单，易于处理.

为什么球面坐标不适用？

根本原因：球面 $x^2+y^2+z^2 = 2z$ 的球心在 $(0,0,1)$，不在原点 $O(0,0,0)$. 因此：

  • 球面不能简单地用 $\rho = \text{常数}$ 或 $\rho = f(\varphi)$ 表示；
  • 需要将区域分割为两部分（球心以上和以下），积分限的确定和计算都非常繁琐；
  • 强行使用球面坐标不仅不能简化计算，反而会大大增加计算量.

方法选择的实用建议

优先选柱面坐标的情形
  • 积分区域在 $xOy$ 平面的投影是圆、圆环或扇形
  • 边界曲面含有 $x^2+y^2$ 或可化为 $r$ 的简单函数
  • 被积函数含有 $x^2+y^2$ 或可化为 $r$ 的函数
  • 区域是圆柱、圆锥或旋转体
  • 典型例子：旋转抛物面与平面围成的区域
优先选球面坐标的情形
  • 积分区域是球体、球锥、球壳（球心在原点）
  • 被积函数含有 $x^2+y^2+z^2 = \rho^2$
  • 边界曲面容易化为 $\rho = f(\varphi)$ 的形式
  • 区域边界由同心球面和锥面构成
  • 典型例子：球面与锥面围成的区域（球心在原点）
核心原则：拿到三重积分题目后，先分析两点：(1) 积分区域是什么形状？(2) 被积函数有什么特点？特别是要判断球心位置——如果球心在原点附近，球面坐标可能是首选；如果球心偏离原点，柱面坐标往往更实用.

积分区域 $\Omega$ 的 3D 可视化

解题要点总结

1
分析区域形状：锥面 $z = \sqrt{x^2+y^2}$ 与球面 $x^2+y^2+(z-1)^2 = 1$ 围成的区域，投影为圆盘.
2
选择坐标系：投影区域是圆形，优先选择柱面坐标.
3
确定积分限：$\theta: 0 \to 2\pi$，$r: 0 \to 1$，$z: r \to \sqrt{2-r^2}$.
4
计算累次积分：先对 $z$ 积分得 $1-r^2$，再对 $r$ 积分得 $\frac{1}{4}$，最后乘 $2\pi$ 得 $\frac{\pi}{2}$.
5
最终结果：$I = \iiint_{\Omega} z\,\mathrm{d}V = \boxed{\dfrac{\pi}{2}}$.
